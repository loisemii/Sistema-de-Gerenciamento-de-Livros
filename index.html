Available Endpoints

POST /api/books/search
Search and save a book by title from Gutendex API
GET /api/books
Get all registered books
GET /api/books/language/{languageCode}
Get books by language (PT, EN, ES, FR)
GET /api/books/{id}
Get a specific book by ID
GET /api/authors
Get all registered authors
GET /api/authors/year/{year}
Get authors who were alive in a specific year
GET /api/authors/{id}
Get a specific author by ID
GET /api/health
Application health check
